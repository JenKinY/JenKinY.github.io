一、创建一个SpringBoot项目

使用 IDEA快速初始化一个 Spring Boot 工程

二、添加依赖

添加 lombok,mysql,mybatis-plus，完成后的代码

<dependencies>
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
        </dependency>

        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-test</artifactId>
            <scope>test</scope>
        </dependency>
        <!-- lombok-->
        <dependency>
            <groupId>org.projectlombok</groupId>
            <artifactId>lombok</artifactId>
            <optional>true</optional>
        </dependency>
        <!-- mybatis-plus -->
        <dependency>
            <groupId>com.baomidou</groupId>
            <artifactId>mybatis-plus-boot-starter</artifactId>
            <version>3.1.0</version>
        </dependency>
        <!-- mysql-->
        <dependency>
            <groupId>mysql</groupId>
            <artifactId>mysql-connector-java</artifactId>
            <version>5.1.21</version>
        </dependency>
    </dependencies>

三、配置

1.修改默认配置文件 application.properties 为 application.yml
2.在 application.yml中添加Mysql相关配置
spring:
  datasource:
    url: jdbc:mysql://127.0.0.1:3306
    username: Mysql登录名
    password: Mysql密码
    driver-class-name: com.mysql.jdbc.Driver
3.在 Spring Boot 启动类中添加 @MapperScan 注解，扫描 Mapper 文件夹：
@SpringBootApplication
@MapperScan("top.skyzc.skyzcmybatisexample.mapper")
public class SkyzcMybatisExampleApplication {

    public static void main(String[] args) {
        SpringApplication.run(SkyzcMybatisExampleApplication.class, args);
    }

}

四、编码

1.新建一个Enity包，编写实体类 User.java
@Data
public class User {
    private Long id;
    private String name;
    private Integer age;
    private String email;
}

@Data为Lombok插件，能自动为标注的类实现 getter/setter 方法，更多Lombok插件的使用方法后续会更新，此处了解即可

2.新建一个mapper包，编写Mapper接口 UserMapper.java
public interface UserMapper extends BaseMapper<User> {

}

五、测试

1.编写测试类
@RunWith(SpringRunner.class)
@SpringBootTest
public class SampleTest {
    @Autowired
    private UserMapper userMapper;

    @Test
    public void testSelect(){
        System.out.println(("----- selectAll method test ------"));
        List<User> userList = userMapper.selectList(null);
        Assert.assertEquals(5,userList.size());
        userList.forEach(System.out::println);
    }
}
